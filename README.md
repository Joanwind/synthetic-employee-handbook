# Synthetic Employee Handbook

This is a synthetic employee handbook created for demonstration purposes.

It is designed to simulate a realistic internal company policy knowledge base
for testing retrieval-augmented generation (RAG) systems.

The content was generated and refined through prompt engineering and structured design, ensuring it reflects realistic workplace policies while maintaining data privacy.

Sections include:
- Company overview
- Remote work policy
- Annual leave
- Sick leave
- Code of conduct
- Performance review
- Benefits and Compensation
- IT and security policy
